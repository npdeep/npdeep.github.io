Where to Measure and How Often

Pradeep Niroula | Sun 16 August 2020

This is another cute problem that I came across recently and did not immediately know the solution to. Suppose I give you a sensor and ask you to measure a quantity along a line (say a magnetic or electric field along a line). I tell you that the function is linear i.e $B(x) = \alpha x + \beta $ with $0 < x < L$. And I aks you to give the exact function by estimating $\alpha$ and $\beta$. Just so you know, the sensor is not perfect — it gives a noisy measurement. Assume that the noise is a simple Gaussian.

If you can only measure, say, a hundred times, what is the best way to do it? You could either take 50 measurements each at two ends $x=0$ and $x=L$? Or you could take 10 measurements each at 10 points between $0$ and $L$. After you take the measurements, you use an ordinary least square fit to get a linear fit.

import numpy as np
import matplotlib.pyplot as plt


def plot_scattered_and_fit(deg, target_poly, n_meas, meas_points):
    xs = []
    ys = []
    for x in np.linspace(0, 1, meas_points):
        for _ in range(int(n_meas/meas_points)):
            xs.append(x)
            ys.append(np.random.normal(target_poly[0]*x + target_poly[1], 0.02))
    plt.scatter(xs, ys, s=2)
    params, cov = np.polyfit(xs, ys, deg=deg, cov=True)
    _xs = np.linspace(0, 1, meas_points)
    xs = np.vstack([_xs**(deg-i) for i in range(deg+1)]).T
    ys = np.dot(xs, params)
    c_i = np.sqrt(np.diag(xs.dot(cov).dot(xs.T)))
    plt.plot(_xs, ys, linestyle="dashed", color="gray")
    plt.fill_between(_xs, ys+c_i, ys-c_i, alpha=0.1, color='red')
    plt.xlim([-0.2, 1.2])
    plt.xlabel("x", fontsize=14)
    plt.ylabel(r"$B(x)$", fontsize=14)

Consider a more general problem. A function $f(x)$ is defined befween $0<x<1$ and takes the functional form of a polynomial of degree $D$. Given $N$ total measurement shots, what is the best way to estimate the parameters of the function? I am not interested in the optimal way of doing it, I just want to see the relation between the total mean square error, and number of measurement locations.

To make the problem a bit mroe formal, assume the function takes the form $$f_{\mathbf{\alpha}} = (x) = \sum_{i=0}^{D-1} \alpha_i x^i \qquad x \in (0, 1)$$ We can do $N$ total measurements. We choose $T$ differnet sites for measurement. Each measurement site gets $N/T$ shots. The measurement data is denoted by the set of pairs $\Pi = \{(y, x)_i\}$. Using these noisy measurement, we estimate the parameters $\hat{\alpha}$ to get the estimated function $\hat{f}_{\hat{\alpha}}$. We compute the mean-square error as the expected error in the range $x = (0, 1)$ $$\mathcal{E} = \left \langle \left(f_\alpha(x) - \hat{f}_{\hat{\alpha}}(x)\right)^2 \right\rangle$$

I numerically compute the estimation error by simulating a thousand random instances of the problem.

def plot_mean_squared_error(deg, n_sims=1000, n_meas=100, min_sites=2, max_sites=5, error=0.02):
    err = np.zeros(max_sites- min_sites+1)
    for i, n_site in enumerate(np.arange(min_sites, max_sites+1)):
        meas_points = int(n_meas/n_site)
        for _ in range(n_sims):
            # assume all the solutions are in the interval [0, 1]
            roots = np.random.uniform(low=0, high=1, size=deg)
            intercept = np.random.uniform(low=0, high=1)
            
            def random_poly(x):
                prod = 1
                for root in roots:
                    prod *= (x - root)
                return prod + intercept
            
            xs = []
            ys = []
            for x in np.linspace(0, 1, meas_points):
                for _ in range(int(n_meas/meas_points)):
                    xs.append(x)
                    ys.append(np.random.normal(random_poly(x), error))
            
            # polyfit
            params, cov = np.polyfit(xs, ys, deg=deg, cov=True)
            
            # compute mean_squared error
            _err = 0
            xs = np.linspace(0, 1, 100)
            for x in np.linspace(0, 1, 100):
                target_y = random_poly(x)
                predicted_y = np.sum([params[i]*(x**(deg-i)) for i in range(deg+1)])
                _err += (target_y - predicted_y) **2 
            _err = _err/100
            err[i] += _err/1000
    return err
